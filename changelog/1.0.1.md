# Version 1.0.1

**Release Date**: December 29, 2025

## üéâ New Features

### Density Scope Extensions
- Added `DensityScreenUtilScope` for property-like syntax with Density scope
- New extension properties `.toH`, `.toW`, and `.toR` within Density scope
- Composable `Density.withScreenUtil()` extension function
- Non-composable `Density.withScreenUtil(state)` for use in LaunchedEffect and other non-composable contexts

**Usage Example:**
```kotlin
@Composable
fun MyScreen() {
    with(LocalDensity.current.withScreenUtil()) {
        val height = 100.toH  // Property-like access
        val width = 50.toW
        val radius = 20.toR
        
        Box(modifier = Modifier.size(width, height))
    }
}
```

### Composable Conversion Functions
- Added `Int.toH()` - Converts raw pixels to height-scaled Dp
- Added `Float.toH()` - Converts raw pixels to height-scaled Dp
- Added `Int.toW()` - Converts raw pixels to width-scaled Dp
- Added `Float.toW()` - Converts raw pixels to width-scaled Dp

- Support for height scaling (`.toH`), width scaling (`.toW`), and radius scaling (`.toR`)

## üöÄ Migration Guide

No migration needed! This release is fully backward compatible. New features are additive and don't affect existing code.

If you want to use the new Density scope extensions:

```kotlin
// Before (still works)
Box(modifier = Modifier.height(100.h))

// New option available
with(LocalDensity.current.withScreenUtil()) {
    Box(modifier = Modifier.height(100.toH))
}
```
